## Features

<Columns cols={2}>
  <Column>
    <Card icon="watch" title="Activity Tracking">
      Automatically track in-game sessions, detect AFK time, and roll up daily stats into heatmaps and leaderboards. See exactly who's active, and for how long.
    </Card>
  </Column>
  <Column>
    <Card icon="file-certificate" title="Courses and Certification">
      Create training programs with articles, videos, PDFs, and quizzes. Assign courses to individuals or entire roles, track completion progress in real-time, and automatically generate certificates.
    </Card>
  </Column>
</Columns>

<Columns cols={2}>
  <Column>
    <Card icon="timer" title="Timeclock">

    </Card>
  </Column>
  <Column>
    <Card icon="book" title="Knowledgebase">
      Write and organize internal documentation with a full rich-text editor supporting tables, images, code blocks, and embedded videos. 
    </Card>
  </Column>
</Columns>

<Columns cols={2}>
  <Column>
    <Card icon="paperclip-vertical" title="Applications">

    </Card>
  </Column>
  <Column>
    <Card icon="key" title="In-Game Management">
