Sessions & Events is your workspace's scheduling hub. Create one-time or recurring events, assign hosts and roles, link to Roblox games, and track attendance automatically.

## Calendar View

Navigate to **Sessions & Events** from the sidebar to see your event calendar. Switch between **month** and **week** views using the toggle at the top.

<Frame>
  <img src="/images/sessions-calendar.png" alt="Sessions calendar view" />
</Frame>

Click any day to see events scheduled for that date. Events are color-coded by type for quick identification.

## Creating an Event

<Steps>
  <Step title="Click Create Event">
    Click the **Create Event** button or click on a day in the calendar.
  </Step>
  <Step title="Fill in event details">
    - **Title** — Name of the event (e.g., "Weekly Training", "Staff Meeting")
    - **Description** — Optional details about the event
    - **Date & Time** — When the event takes place
    - **Event Type** — Training, meeting, interview, patrol, or custom type
  </Step>
  <Step title="Link a Roblox game">
    Select a tracked Roblox experience to link the event to. Participants will see the game icon, name, and a one-click **Join** button on the event detail page.
  </Step>
  <Step title="Save">
    Click **Create** to add the event to the calendar.
  </Step>
</Steps>

## Assigning Hosts and Roles

Every event can have assigned staff:

### Host & Co-Host

- **Host** — The primary person running the event
- **Co-Host** — A secondary leader assisting the host

To assign, edit the event and select users from the **Host** and **Co-Host** dropdowns.

### Custom Session Roles

For more complex events, create custom roles:

1. Edit the event and find the **Roles** section
2. Click **Add Role** to create a position (e.g., "Trainer", "Supervisor", "Observer")
3. Assign members to each role

<Frame>
  <img src="/images/sessions-roles.png" alt="Session role assignments" />
</Frame>

<Tip>
  Create role templates in your event type settings to pre-configure staffing for each type of event. This saves time when creating similar events repeatedly.
</Tip>

## Recurring Events

Set events to repeat on a schedule so you don't have to create them each time.

<Steps>
  <Step title="Enable recurring">
    When creating or editing an event, toggle **Recurring** on.
  </Step>
  <Step title="Choose frequency">
    Select how often the event repeats:
    - **Daily**
    - **Weekly**
    - **Biweekly** (every 2 weeks)
    - **Monthly**
    - **Weekdays** (Monday through Friday)
  </Step>
  <Step title="Set an end date (optional)">
    Choose when the recurring series should stop. Leave blank for indefinite recurring.
  </Step>
</Steps>

<Note>
  Editing a recurring event will ask whether you want to update just this occurrence or all future occurrences.
</Note>

## Server Lock & Unlock

Control when members can join the game server for an event:

- **Unlock Window** — How many minutes before the event start time the server opens for joining
- **Lock Window** — How many minutes after the event starts before the server locks to new joins

This prevents early arrivals or late entries and keeps events structured.

## Viewing Event Details

Click on any event in the calendar to see its detail page:

- Event title, description, and time
- Linked Roblox game with join button
- Assigned host, co-host, and role assignments
- Attendance data (if game tracking is set up)

## Managing Events

- **Edit** — Click the edit button on any event to modify details, roles, or recurring settings
- **Delete** — Remove an event. For recurring events, choose to delete just this occurrence or the entire series
- **Filter** — Use the event type filter to show only specific kinds of events on the calendar

<Tip>
  Session attendance feeds into the [Leaderboard](/using-roteam/leaderboard) when using the "Top Sessions" mode, so hosting and assisting in events helps members climb the rankings.
</Tip>
